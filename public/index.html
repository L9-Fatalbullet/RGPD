Étapes majeures de conformité

1. Planification

  • Identifier les traitements de données
  • Nommer un responsable
  • Évaluer les besoins de conformité
Bonne pratique : Impliquer la direction dès le début.
Avertissement : Négliger cette étape peut entraîner des sanctions.
Exemple : Ministère public planifiant la collecte de données citoyennes.
[Diagramme ici]

2. Collecte

  • Informer les personnes concernées
  • Obtenir le consentement
  • Limiter la collecte au nécessaire
Bonne pratique : Utiliser des formulaires clairs.
Avertissement : Collecte excessive interdite.
Exemple : Site e-commerce collectant des emails clients.
[Diagramme ici]

3. Stockage

  • Stocker de façon sécurisée
  • Chiffrer les données sensibles
  • Limiter l'accès
Bonne pratique : Utiliser des serveurs au Maroc.
Avertissement : Fuite de données = obligation de notification.
Exemple : Clinique stockant des dossiers médicaux.
[Diagramme ici]

4. Maintenance

  • Mettre à jour les mesures de sécurité
  • Former le personnel
  • Contrôler les accès régulièrement
Bonne pratique : Audits réguliers.
Avertissement : Négligence = responsabilité du responsable.
Exemple : Hôpital mettant à jour ses accès informatiques.
[Diagramme ici]

5. Application des droits

  • Permettre l'accès, la rectification, l'opposition
  • Répondre dans les délais légaux
  • Documenter les demandes
Bonne pratique : Procédure écrite pour les demandes.
Avertissement : Refus injustifié = sanction CNDP.
Exemple : Université traitant une demande d'accès étudiant.
[Diagramme ici]

6. Archivage

  • Définir une durée de conservation
  • Détruire ou anonymiser à l'échéance
  • Tenir un registre des destructions
Bonne pratique : Politique d'archivage claire.
Avertissement : Conservation excessive interdite.
Exemple : Société supprimant les CV après recrutement.
